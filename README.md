|---tests                     Test Cases located to execute

## Prerequisites
1. Node.js
2. Chrome Browser

## Project Setup 
1. Clone this repository 
2. Go to the repository Folder 
3. Run ```npm install``` in the terminal
4. Create your .env file with the following 

    EMAIL=example@mail.com
    PASSWORD=password
    URL=Todoist.com
